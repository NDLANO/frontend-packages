# eslint-config-ndla

Overrides some settings to better suit NDLA.

## Installing

```bash
# Install
yarn add --save-dev eslint-config-ndla
```

```bash
# Install
npm install --save-dev eslint-config-ndla
```

## Usage

### Configs

Edit your `.eslintrc` file and extend the config.

Example:

```javascript
// .eslintrc
{
  "extends": "eslint-config-ndla"
}
```
